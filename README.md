
# Snake Game

A basic snake game made by Pygame Module in Python.
## Difficulty Value
 
If you want to set Difficulty value to 10, you can set it like difficulty = 10

- Easy       -  10
- Medium     -  25
- Hard       -  40
- Extreme    -  60
- Impossible -  120
